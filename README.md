# MVP_exam

Простое веб‑приложение для отслеживания привычек и личных целей. Пользователь может создавать привычки, отмечать выполнение по дням и смотреть простую статистику (процент выполнения, серии и календарь).

## Стек технологий

- HTML (index-2.html) [file:3]
- CSS (style.css) [file:2]
- Чистый JavaScript без сторонних библиотек (встроен в index-2.html) [file:3]

## Требования

- Любой современный браузер (Chrome, Firefox, Edge, Safari).
- Локальные файлы: `index-2.html` и `style.css` должны находиться в одной папке. [file:2][file:3]

## Запуск приложения (вариант 1 — самый простой)

1. Скачайте или склонируйте репозиторий к себе на компьютер.
2. Убедитесь, что файлы `index-2.html` и `style.css` лежат в одной директории. [file:2][file:3]
3. Откройте файл `index-2.html` двойным щелчком мыши или через пункт «Открыть с помощью…» в браузере. [file:3]
4. Приложение откроется в браузере и будет работать полностью локально (данные сохраняются в `localStorage` браузера). [file:3]

## Запуск через простой локальный сервер (рекомендуется)

Для корректной работы в некоторых браузерах может потребоваться запуск через локальный сервер (особенно при доработке кода).

### Вариант с Python

1. Откройте терминал / командную строку.
2. Перейдите в папку проекта:
   - `cd путь/к/папке/проекта`
3. Запустите локальный сервер:
   - для Python 3: `python -m http.server 8000`
4. Откройте в браузере адрес: `http://localhost:8000/index-2.html`. [file:3]

### Вариант с VS Code (Live Server)

1. Откройте папку проекта в VS Code.
2. Установите расширение **Live Server**.
3. Откройте `index-2.html` и нажмите «Open with Live Server». [file:3]
4. Браузер откроет приложение по локальному адресу (обычно `http://127.0.0.1:5500/...`).

## Основные возможности

- Создание привычки: название, описание, периодичность (ежедневно, еженедельно, выбор дней недели), время напоминания. [file:3]
- Просмотр списка привычек с фильтрами: все, на сегодня, выполненные. [file:3]
- Отметка выполнения привычки за текущий день одним кликом. [file:3]
- Просмотр статистики по привычке: процент выполнения, текущая и лучшая серия, общее количество выполнений, календарь за последние 30 дней. [file:3]
- Редактирование и удаление привычек. [file:3]

## Структура проекта

- `index-2.html` — разметка страницы и бизнес‑логика трекера привычек (JavaScript внутри файла). [file:3]
- `style.css` — стили приложения (цвета, сетки, адаптивная верстка, оформление карточек и модальных окон). [file:2]

## Сохранение данных

- Все данные (список привычек, выполнение по дням) сохраняются в `localStorage` браузера под ключом `habitTrackerData`. [file:3]
- Данные не отправляются на сервер и доступны только на том устройстве и в том браузере, где вы запускали приложение. [file:3]

## Известные ограничения MVP

- Нет регистрации и синхронизации между устройствами. [file:3]
- Напоминания реализованы только как поле времени внутри привычки, без реальных пуш‑ или email‑уведомлений. [file:3]
